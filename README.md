# HerHealthHormone
Designing and implementing a user-friendly website which tracks user symptoms and data to help alleviate PCOS symptoms. 

## Features 
✧ Integrated AI (LLAMA) Which takes user input regarding name, mood, sleep hours, diet, stress, and symptoms. 

✧ Takes user input and generates ICS file output.

✧ ICS file can be uploaded onto Google Calendar for easy access.

✧ Daily reminders and optmized outputs which alleviates PCOS symptoms.
